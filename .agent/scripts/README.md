# Scripts - Local Validation

## Core scripts
- `checklist.py`: priority validation checks
- `verify_all.py`: full validation suite
- `session_manager.py`: state/session utility
- `auto_preview.py`: local preview helper

## Typical local usage
```powershell
python .agent/scripts/checklist.py .
python .agent/scripts/verify_all.py . --url http://localhost:3000
```
