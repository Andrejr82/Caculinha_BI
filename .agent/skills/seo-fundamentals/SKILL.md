# Fundamentos de SEO

> Princípios para visibilidade em mecanismos de busca.

---

## 1. Framework E-E-A-T

| Princípio | Sinais |
|-----------|--------|
| **Experiência** | Conhecimento de primeira mão, exemplos reais |
| **Expertise** | Credenciais, profundidade de conhecimento |
| **Autoridade** | Backlinks, menções, reconhecimento da indústria |
| **Confiança** (Trust) | HTTPS, transparência, informações precisas |

---

## 2. Core Web Vitals

| Métrica | Alvo | Mede |
|---------|------|------|
| **LCP** | < 2.5s | Performance de carregamento |
| **INP** | < 200ms | Interatividade |
| **CLS** | < 0.1 | Estabilidade visual |

---

## 3. Princípios de SEO Técnico

### Estrutura do Site

| Elemento | Propósito |
|----------|-----------|
| Sitemap XML | Auxiliar o rastreamento (crawling) |
| robots.txt | Controlar o acesso |
| Tags Canônicas | Prevenir conteúdo duplicado |
| HTTPS | Sinal de segurança |

### Performance

| Fator | Impacto |
|-------|---------|
| Velocidade da página | Core Web Vital |
| Amigável ao mobile | Fator de ranking |
| URLs limpas | Rastreabilidade |

---

## 4. Princípios de SEO de Conteúdo

### Elementos da Página

| Elemento | Melhor Prática |
|----------|----------------|
| Title tag | 50-60 carac., palavra-chave no início |
| Meta description | 150-160 carac., persuasivo |
| H1 | Um por página, palavra-chave principal |
| H2-H6 | Hierarquia lógica |
| Alt text | Descritivo, sem excesso de palavras-chave |

### Qualidade do Conteúdo

| Fator | Importância |
|-------|-------------|
| Profundidade | Cobertura abrangente |
| Recência (Freshness)| Atualizações regulares |
| Unicidade | Valor original |
| Legibilidade | Escrita clara |

---

## 5. Tipos de Schema Markup

| Tipo | Uso |
|------|-----|
| Article | Posts de blog, notícias |
| Organization | Informações da empresa |
| Person | Perfis de autor |
| FAQPage | Conteúdo de perguntas e respostas |
| Product | E-commerce |
| Review | Avaliações |
| BreadcrumbList| Navegação (trilha de migalhas) |

---

## 6. Diretrizes para Conteúdo de IA

### O Que o Google Procura

| ✅ FAÇA | ❌ NÃO FAÇA |
|---------|-------------|
| Rascunho de IA + edição humana | Publicar conteúdo bruto de IA |
| Adicionar insights originais | Copiar sem agregar valor |
| Revisão de especialista | Pular a verificação de fatos |
| Seguir o E-E-A-T | Excesso de palavras-chave (stuffing) |

---

## 7. Fatores de Ranking (Priorizados)

| Prioridade | Fator |
|------------|-------|
| 1 | Conteúdo de qualidade e relevante |
| 2 | Backlinks de sites de autoridade |
| 3 | Experiência de página (Core Web Vitals) |
| 4 | Otimização para dispositivos móveis |
| 5 | Fundamentos de SEO técnico |

---

## 8. Medição

| Métrica | Ferramenta |
|---------|------------|
| Rankings | Search Console, Ahrefs |
| Tráfego | Analytics |
| Core Web Vitals| PageSpeed Insights |
| Indexação | Search Console |
| Backlinks | Ahrefs, Semrush |

---

> **Lembre-se:** SEO é um jogo de longo prazo. Conteúdo de qualidade + excelência técnica + paciência = resultados.
